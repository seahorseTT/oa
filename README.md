# oa
my_oa 
自己写的一个小报销单系统，除了基本的CRUD操作，还有个报销单的处理流程。员工填写报销单，部门经理审核，若超过规定金额，部门经理再提交总经理审核，审核通
过后才能提交财务打款，过程中的所有操作及操作时间都会记录在报销单处理明细中。
    等待完善的功能：员工的查找，可以通过搜索员工编号或姓名（模糊查询）直接查找到员工信息

               员工登录成功，只能在左上角显示名称和职位，头像无法自定义（数据库暂时没这玩意）
